## Notas

- Os adapters estão localizados em `app/Adapters/`
- Para ajustar a implementação conforme a documentação oficial, edite os adapters mantendo a interface `SubadquirerInterface`
- As URLs base são configuradas através das variáveis de ambiente no `.env`
